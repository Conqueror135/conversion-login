## Messenger login
1. Nhúng link cdn vào file html
```
<script src="https://cdn.jsdelivr.net/gh/Conqueror135/conversion-login@latest/messenger-login.js"></script>

```
2. Thiết lập id cho button đăng ký là: messenger-login-btn
